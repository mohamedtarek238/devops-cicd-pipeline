# DevOps CI/CD Pipeline with Jenkins and Docker

## Project Overview

This project demonstrates the implementation of a complete CI/CD pipeline using Jenkins and Docker on CentOS Stream 10.

## Objectives

- Automate application deployment
- Build Docker images automatically
- Deploy containers using Jenkins
- Document the entire implementation process
